# Tests

Thư mục này chứa các test files cho dự án PHTV.

## Nội dung

- `IntegrationTests.swift` - Integration tests cho PHTV
- `PHTVFunctionalTests.swift` - Functional tests cho các tính năng chính

## Chạy tests

```bash
# Mở project trong Xcode
open PHTV.xcodeproj

# Chạy tests bằng Xcode
# Product > Test (⌘U)
```

## Lưu ý

Các test files này hiện chưa được tích hợp vào Xcode project. Để sử dụng, cần add vào test target trong Xcode.

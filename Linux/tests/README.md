# Linux tests

Khu vực dự kiến chứa:

- IBus integration test host;
- Fcitx 5 addon harness;
- GTK/libadwaita UI tests;
- Wayland/X11 application smoke tests;
- `.deb`/`.rpm` install, upgrade và uninstall tests.

Golden vectors engine nằm tại `Shared/TestVectors`, không copy vào Linux.
Chiến lược đầy đủ ở [../docs/TESTING.md](../docs/TESTING.md).

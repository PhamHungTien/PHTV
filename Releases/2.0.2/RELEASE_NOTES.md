# PHTV 2.0.2 Release Notes

## New Features

- **Onboarding System Settings Step**: Thêm bước hướng dẫn cài đặt hệ thống trong quy trình Onboarding
  - Hướng dẫn trực quan để tắt các tính năng tự động sửa lỗi của macOS (auto-correction, smart quotes, etc.)
  - Thêm hình ảnh minh họa cài đặt Input Source trong System Settings
  - Giúp người dùng mới tránh xung đột giữa PHTV và macOS input features

- **Image Zoom in Onboarding**: Thêm khả năng phóng to ảnh hướng dẫn
  - Click vào ảnh để xem phóng to
  - Icon kính lúp để người dùng biết có thể zoom
  - Đóng bằng nút close hoặc phím Escape

- **Help Button in Settings**: Thêm nút '?' vào thanh công cụ Settings
  - Cho phép người dùng xem lại Onboarding bất cứ lúc nào
  - Hữu ích khi cần xem lại hướng dẫn cài đặt hệ thống

## Documentation

- Cập nhật README.md và INSTALL.md với cảnh báo về việc tắt các tính năng tự động sửa lỗi của macOS trước khi sử dụng PHTV
